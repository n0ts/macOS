OS X
===================================
Shell scripts for automated OS X machine setup.


Requirements
------------
### Platforms
- OS X 10.9 Mavericks


Usage
-----

  1. Open the Terminal.app

  2. Run follow command

```
  export HOSTNAME=<hostname>
  curl -Ls https://raw2.github.com/n0ts/os-x/master/00-xcode.sh | bash -
```
